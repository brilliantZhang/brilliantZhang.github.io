这周的任务是搭建wordpress博客，之前也考虑过用wordpress,但感觉配置起来要比hexo+github pages要麻烦些，就没有尝试．

LAMP环境部署

Linux Apache Mysql PHP
PHP：超文本预处理器 是一种开源的通用计算机脚本语言
Apache： Web服务器,负责处理请求，并将对PHP的请求转发给PHP执行。还有另一种web服务器叫Nginx.

安装 Apache2

在终端输入该命令 ，使用 apt-get 安装 Apache2：
sudo apt-get install apache2 -y　

安装PHP 组件

sudo apt-get install php7.0 -y
sudo apt-get install libapache2-mod-php7.0

安装 MySQL 服务

sudo apt-get install mysql-server -y
sudo apt-get install php7.0-mysql

安装 phpmyadmin

sudo apt-get install phpmyadmin -y
sudo ln -s /usr/share/phpmyadmin /var/www/html/phpmyadmin
sudo service mysql restart -y
sudo systemctl restart apache2.service

安装并配置 Wordpress

安装 Wordpress

wget https://cn.wordpress.org/wordpress-4.7.4-zh_CN.zip
sudo unzip wordpress-4.7.4-zh_CN.zip

为 wordpress 配置一个数据库

mysql -u root -p
CREATE DATABASE wordpress;
CREATE USER wordpressuser;
SET PASSWORD FOR wordpressuser= PASSWORD(“password123”);
GRANT ALL PRIVILEGES ON wordpress.* TO wordpressuser IDENTIFIED BY”password123”;
FLUSH PRIVILEGES;
exit;

配置 wordpress

sudo mv wordpress/* /var/www/html/
sudo chmod -R 777 /var/www/html/
sudo mv /var/www/html/index.html /var/www/html/index~.html
sudo systemctl restart apache2.service

测试访问

访问服务器主机ip地址,可进行wordpress服务的安装．IP地址可在终端输入命令
ifconfig -a 获得．

linux学习（二）：打包

在Debian系下，通常打包为deb包。可以用checkin stall，也可用dpkg-deb命令。

dpkg-deb

在源代码目录新建一个DEBIAN目录，再新建一个control文本，位置一定要对，否则不能打包成功。编辑control文本，输入以下信息：

Package: 
Version: 
Section: 
Priority:
Depends:
Suggests:
Architecture: 
Installed-Size: 
Maintainer: 
Provides:
Description:

再执行命令:

$ sudo dpkg-deb -b dir packagename
