python实战：爬取教务在线学生照片

发表于 2018-01-27 | 更新于 2019-08-04

写个爬虫来爬取照片作为练手项目，采用python语言实现，用requests库获取html内容和用BeautifulSoup4库匹配提取网页内容。

具体代码如下：

import requests
from bs4 import BeautifulSoup
import os
import time
import random
import re



UA=["Mozilla/5.0 (Windows NT 6.1; WOW64) AppleWebKit/537.1 (KHTML, like Gecko) Chrome/22.0.1207.1 Safari/537.1",
 "Mozilla/5.0 (X11; CrOS i686 2268.111.0) AppleWebKit/536.11 (KHTML, like Gecko) Chrome/20.0.1132.57 Safari/536.11",
 "Mozilla/5.0 (Windows NT 6.1; WOW64) AppleWebKit/536.6 (KHTML, like Gecko) Chrome/20.0.1092.0 Safari/536.6",
 "Mozilla/5.0 (Windows NT 6.2) AppleWebKit/536.6 (KHTML, like Gecko) Chrome/20.0.1090.0 Safari/536.6",
 "Mozilla/5.0 (Windows NT 6.2; WOW64) AppleWebKit/537.1 (KHTML, like Gecko) Chrome/19.77.34.5 Safari/537.1",
 "Mozilla/5.0 (X11; Linux x86_64) AppleWebKit/536.5 (KHTML, like Gecko) Chrome/19.0.1084.9 Safari/536.5",
 "Mozilla/5.0 (Windows NT 10.0; Win64; x64; rv:57.0) Gecko/20100101 Firefox/58.0"]

href=''
ID=''

header={"User-Agent":random.choice(UA)}
print("选择输入班级或学号:（1）学号（2）班级")
choice=input()

if choice == '1':
    print("输入学号:")
    ID=input()

    htmlstudent="http://jwzx.cqupt.edu.cn/showstuPic.php?xh="+ID
    html=htmlstudent
    print("正在爬取"+ID+"的照片")
    image=requests.get(html)
    try:
        name=html[-10:]
        with open (name+'.jpg','ab')as f:
            f.write(image.content)
            f.close()
    except:
        pass


else:
    print("输入班级:")
    href=input()
    htmlclass="http://jwzx.cqupt.edu.cn/jwzxtmp/showBjStu.php?bj="+href
    html=htmlclass

    r=requests.get(html,headers=header)
    soup=BeautifulSoup(r.content,'html.parser')
    tr_list=soup.find_all('td',class_='pic')



    for img in tr_list:
        img_list=img.find_all('img')
        for url in img_list:
            try:
                src=url['src']
                image=requests.get(src)
                name=src[-10:]
                print("正在爬取"+href+name+"的照片")
                with open (name+'.jpg','ab')as f:
                    f.write(image.content)
                    f.close()
            except:
                pass

通过写这段爬虫代码，简单回顾了一下python的循环语句和抛出异常语句以及切片操作和文件操作。其中，切片操作有个坑，在切片结束位置处的元素不能取到，若要完整取到最末尾元素，则要缺省该值。

大致思路是用requests库get方法获取html信息，再用BeautifulSoup库find_all方法匹配标签内容或其属性值，再通过文件操作写入。

另外，可以使用通过降低速度，使用IP代理池应该可以避开反爬虫机制。
